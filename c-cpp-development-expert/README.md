# C/C++ 开发专家

一个全面的 C/C++ 开发辅助 Skill，提供从语言特性到项目架构的完整知识体系，帮助开发者高效完成 C/C++ 项目开发、代码重构、性能优化等任务。

## 核心技能

### 1. 语言特性支持
- **C 语言全版本支持**：C89/C90、C99、C11、C17、C23 标准特性详解
- **C++ 全版本支持**：C++98、C++11、C++14、C++17、C++20、C++23 标准特性
- **版本迁移指导**：老旧代码升级到现代 C/C++ 的策略和步骤
- **最新特性应用**：智能指针、Lambda 表达式、协程、Concepts、Ranges 等

### 2. 项目架构设计
- **项目结构规划**：根据项目规模设计合理的目录结构和模块划分
- **构建系统配置**：CMake 和 Makefile 配置与优化
- **依赖管理**：第三方库选型、版本兼容性管理
- **设计模式应用**：结合 C/C++ 特性的设计模式选择和实现
- **模块化设计**：代码组织和接口设计最佳实践

### 3. 编码规范与最佳实践
- **命名规范**：C/C++ 命名约定和风格指南
- **代码风格**：缩进、注释、格式化标准
- **内存管理**：RAII、智能指针、资源管理最佳实践
- **错误处理**：异常处理、错误码、std::optional 使用
- **安全性规范**：防止缓冲区溢出、整数溢出、悬空指针
- **性能优化**：移动语义、避免不必要的拷贝、算法优化

### 4. 标准库精通
- **C 标准库**：完整的头文件分类和函数参考
- **C++ 容器库**：vector、map、unordered_map、array 等
- **算法库**：排序、查找、变换、并行算法
- **智能指针与内存**：unique_ptr、shared_ptr、weak_ptr
- **并发与同步**：thread、mutex、atomic、future
- **输入输出**：iostream、filesystem、stringstream
- **字符串处理**：string、string_view、regex
- **现代 C++ 特性**：optional、variant、any、tuple
- **C++20 新增**：ranges、format、coroutines

### 5. 版本兼容性管理
- **版本特性查询**：每个标准的语言特性和新增库对照表
- **编译器支持情况**：GCC、Clang、MSVC 版本对照
- **特性检测方法**：预定义宏、静态断言、在线查询
- **版本迁移指南**：从旧标准升级到现代标准的步骤
- **跨平台兼容性**：平台差异处理、字节序问题

### 6. 设计模式与惯用法
- **创建型模式**：单例、工厂、构建器
- **结构型模式**：适配器、装饰器、代理
- **行为型模式**：观察者、策略、命令
- **C++ 惯用法**：RAII、Pimpl、CRTP、类型擦除
- **现代 C++ 模式**：类型安全的联合、期望值模式、协程模式
